# 从网页图片提取化工产业链信息技能

## 触发条件
- 需要从财经文章图片中提取化工产业链图谱信息
- 需要补充完善化工大宗商品知识图谱
- 需要从新闻资讯图片提取结构化产业链数据

## 流程步骤

### 1. 文章获取与图片提取
1. 使用 `web_extract` 获取目标网页内容
2. 从HTML中提取所有图片URL，筛选出标题符合"化工产业链图谱"的图片
3. 按文章顺序排列图片

### 2. 视觉识别提取信息
1. 对每张可访问的图片使用 `vision` 工具进行视觉分析
2. 要求提取完整产业链路径、节点关系、百分比占比、原料消耗数据
3. 逐条记录识别结果，保留原始结构

### 3. 准确性复核（关键步骤）
1. 搜索网络上是否有其他网站转载同一篇文章
2. 优先找有文字版描述的转载文章（搜狐、网易等门户通常会转载）
3. 使用 `web_extract` 提取文字版内容
4. 逐条对比图片识别结果和文字版描述，验证准确性
   - 标记复核通过/不通过
   - 修正识别错误

### 4. 整合入库
1. 读取现有产业链CSV文件（通常是 `commodity_chain.csv`）
2. 将复核后的信息整合补充，新增富集版文件 `commodity_chain_enriched.csv`
   - 保留原有数据不动
   - 补充新增上下游产品节点
   - 添加消费占比等量化数据
3. 将完整提取结果保存为Markdown文档存档

### 5. Git提交
提交所有新增文件到wiki仓库，提交信息包含"数据补充：新浪化工产业链图谱整合"

## 注意事项
- 部分图片可能404，跳过不可访问的图片，只提取可访问的
- 文章标题说"21条"不代表网页真有21张图片，实际以网页展示为准
- 图片识别结果必须经过文字版复核才能入库，保证准确性
- 如果识别结果有差异，以文字版描述为准进行修正

## 示例输出
- 新浪财经21条化工产业链图谱提取.md - 完整提取记录
- commodity_chain_enriched.csv - 整合后的结构化数据
